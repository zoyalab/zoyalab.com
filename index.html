Join us

Python / Django web application developer

We currently have a position opened for talented programmer with strong experience in building web apps using Django framework.

We offer the full-time employment in the office located in Zielona Góra, Poland, competetive salary, flexible working hours and - most importantly - opportunity to expand working experience around international web projects of various scope.

We need you to:

  • have experience building web applications using Django, obviously;
  • know how to use Git;
  • have at least basic experience with Linux command line interface.

Nice-to-haves:

  • evidence of programming passion, ie. GitHub account with own contributions, samples of involvment to non-work related projects etc.;
  • good working experience with LAMP stack (although we love Django, we still do lots of PHP-based stuff too);
  • experience with front-end technologies like HTML5, CSS3, JavaScript, jQuery, node.js etc.;
  • using Linux desktop environment on daily basis.
